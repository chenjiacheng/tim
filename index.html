简介

Tim

腾讯云 即时通信 IM SDK for PHP

Laravel 扩展包：传送门

Hyperf 扩展包：传送门

运行环境

  • PHP >= 8.0.2
  • Composer >= 2.0

相关文档

Tencent IM

安装方式

composer require chenjiacheng/tim

使用示例

// 配置信息
$config = [
    'sdkappid'   => '1400000000',
    'key'        => 'd182df719a269501ec4795f980aa3691cae60412335058c161c3467d3cb0f565',
    'identifier' => 'administrator',
];

// 实例化对象
$tim = new Tim($config);
